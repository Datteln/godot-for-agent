## 3. 字段对照表（协议权威定义）

### 3.1 HTTP 端点

| 端点 | 方法 | 请求 | 响应 | 鉴权 |
|------|------|------|------|------|
| `/chat` | POST | ChatRequest（§3.2） | 三态（§3.5） | Bearer token |
| `/health` | GET | — | `{ok, model, endpoint_reachable, function_calling_supported}` | Bearer token |
| `/reset` | POST | `{"session_id": "..."}` | `{ok}` | Bearer token |
| `/skills` | GET | — | `[{qualified_name,name,description,when_to_use,source,enabled,effective_tools,warnings}]` | Bearer token |
| `/doctor` | GET | — | 安装/LLM/MCP/索引/LSP/Memory/权限/扩展/上下文自检 | Bearer token |
| `/commands` | GET | — | typed command + SkillCommand 清单 | Bearer token |
| `/commands/{name}` | POST | command 参数 | command 执行结果/后台 task 状态 | Bearer token |
| `/memory` | GET/POST | 查询/保存/删除动作 | Memory 状态/结果 | Bearer token |
| `/recovery-pointer` | GET | `project_root` | 最小恢复指针 | Bearer token |

### 3.2 ChatRequest（前端 → 服务）

| 字段 | 类型 | 必填 | 前端来源 | 服务端用途 |
|------|------|------|------|------|
| `session_id` | str | ✅ | `AgentHttpClient.session_id`（uuid） | 会话定位 + per-session 锁 |
| `request_id` | str | 用户消息时 | `_new_uuid()` | 用户消息级幂等键 |
| `user_message` | str \| null | 二选一 | 输入框 | 新一轮用户消息 |
| `tool_results` | ToolResult[] \| null | 二选一 | 工具执行/确认结果 | resume 挂起帧 |
| `context` | Context \| null | 可选 | `ContextCollector.collect()` | 结构化上下文（§3.3） |
| `language_hint` | str \| null | 可选 | `selection.script_language` | `gdscript` / `csharp` 语言约束 |
| `engine_version` | str \| null | 可选 | `Engine.get_version_info()` → `"4.4.1"` | 选 doc dump 版本 |
| `permission_mode` | str \| null | 可选 | EditorSettings（机器级，**不读 ProjectSettings**） | 权限模式（信任模型再钳制） |
| `effort` | str \| null | 可选 | EditorSettings `ai_agent/effort` | quick/standard/deep/verify/advisor |
| `output_style` | str \| null | 可选 | EditorSettings `ai_agent/output_style` | OutputStyle id，只影响表达方式 |

### 3.3 Context（结构化上下文）

| 字段 | 类型 | 前端采集 | 备注 |
|------|------|------|------|
| `selection` | dict | 选中节点 + 挂载脚本（path/content/language） | `script_language: "gdscript"/"csharp"` |
| `scene_tree` | dict | 递归限深 5 | name/class/path/script/children |
| `tile_catalog` | list | 选中 TileMapLayer 的 TileSet | `{source_id, atlas_coords:[x,y], name}`（上下文用 atlas_coords 数组；**工具参数用 atlas_x/atlas_y**） |
| `project_files` | list | `.gd/.cs/.tscn/.tres`，限 200、跳过 addons | 前端比服务端 deny_read 更严，可接受 |
| `debugger_errors` | list | M1 空；M2 接 EditorDebuggerPlugin | `{type, message, file, line}` |
| `diagnostics` | list | 调试器/LSP 诊断 | `{path,line,column,severity,code,message,source}`，服务端按相关性 Top-N 注入 |
| `dotnet_enabled` | bool | 检测 `.csproj` | 服务端据此暴露/隐藏 C# 工具（PRD D2） |

### 3.4 ToolResult（前端 → 服务，每条）

| 字段 | 类型 | 填写方 | 说明 |
|------|------|------|------|
| `tool_use_id` | str | 服务下发原样回传 | 必须 ∈ 该轮 `pending_tool_call_ids` |
| `frame_id` | str | 服务下发原样回传 | resume 到对应 agent 帧 |
| `turn_id` | str | **AgentHttpClient 统一填**（`current_turn_id`） | 不匹配 → 服务端忽略（幂等） |
| `status` | `"applied" / "rejected" / "error"` | 前端 | rejected/error 同样回传供模型调整 |
| `result` | JSON 值 \| null | 前端工具返回 | **非裸字符串**；写文件需含 `before_hash/after_hash/mtime_ns/known_full_read`；`read_class_docs` 返回含 `source` 的签名 dict |
| `error_code` | str \| null | 前端 | 如 `PATH_DENIED` / `NO_SCRIPT` / `UNKNOWN_TOOL` / `NOT_IMPLEMENTED_M2` / `CLASS_NOT_FOUND` |
| `artifact_refs` | str[] | 前端 | 落地产物 `res://` 路径 |
| `grant_session_allow` | bool | 前端（M2 勾选） | 授权升级；高风险工具（`run_tests`/`set_project_setting`/`batch_rename`）服务端忽略 |

### 3.5 服务响应三态

| `type` | 字段 | 前端处理 |
|------|------|------|
| `tool_calls` | `turn_id`、`text?`、`calls[]`：`{id, name, input, needs_confirm, frame_id, agent, render_kind}` | 存 `current_turn_id`；按 `needs_confirm` 分流静默执行/预览确认；按 `render_kind` 渲染 |
| `final` | `text` | 展示 + `commit_batch()` + 清空 `current_turn_id` |
| `error` | `text` | 展示 + `abort_batch()` + 清空 `current_turn_id` |

### 3.6 枚举值统一

| 枚举 | 取值 | 出处 |
|------|------|------|
| 工具类型（PRD 口径） | 只读 / 改动 / 执行 | PRD §5.5 |
| effect（实现口径） | `read_project` / `write_project` / `execute_project` / `network` / `egress_code` | 主方案 §7、详设A §3.8 |
| 权限决策 | `allow` / `ask` / `deny` | 详设A §3.1 |
| 权限模式 | `default` / `plan` / `auto_approve` / `read_only` | 详设A §3.3 |
| effort | `quick` / `standard` / `deep` / `verify` / `advisor` | 主方案 §6.5 |
| ToolResult.status | `applied` / `rejected` / `error` | 主方案 §14 |
| 签名来源 | `ClassDB` / `script_class` / `unknown` | 前端 §3.5；`enrich` 仅对 `ClassDB` 补 prose |
| Agent/Skill/OutputStyle 来源 | `bundled` / `user` / `project` / `plugin` | 详设B §2.3、主方案 §9.2；project/plugin 未信任前不可提权 |
| render_kind | `diff` / `list` / `map` / `run` / `log` / `json` | 前端预览渲染 |

### 3.7 工具清单（名称 × side × effect × 默认决策）

| 工具 | domain | side | effect | default 模式 |
|------|------|------|------|------|
| `read_script` / `read_scene_tree` / `read_debugger_errors` / `read_class_docs` / `read_project_settings` | 各域 | front | read | allow（静默） |
| `list_files`(前端) / `list_project_files` | program | front | read | allow |
| `list_files` / `grep_code` / `search_codebase` / `load_skill` | program/core | server | read | allow |
| `propose_script_edit` / `propose_tests` | program | front | write | ask |
| `fill_rect` / `draw_rect_border` / `draw_line` / `set_cells` / `clear_rect` | map | front | write | ask |
| `add_node` / `set_node_property` / `instance_scene` | scene | front | write | ask |
| `create_resource` / `batch_rename` / `set_project_setting` | resource | front | write | ask（后两者**禁"总是允许"**） |
| `run_tests` | program | front | **execute** | ask（仅受信任工程；timeout/cancel/日志；**禁"总是允许"**） |
| `delegate` | core | 编排层 | — | 不过权限闸；**单独成轮** |

> 地图工具参数：`fill_rect/draw_rect_border/clear_rect` → `x,y,width,height`；`draw_line` → `from_x,from_y,to_x,to_y`；瓦片 → `source_id, atlas_x, atlas_y`。`propose_script_edit` → `path, before?, after`（或 `diff[]`）。

### 4.D 校对方法备忘

新增/修改任一文档时，按以下顺序快速复检：

1. **字段**：改动是否触及 §3 协议表？→ 同步两侧 DTO + 本表。
2. **枚举**：新值是否进 §3.6？
3. **工具**：新工具是否进 §3.7（side/effect/默认决策）并归 agent？
4. **安全**：是否新增写路径/执行/网络行为？→ 对照详设B 六道边界 + 详设A 权限基线。
5. **版本**：升被改文档版本号 + 变更行，并更新本表 §1。
